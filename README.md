# AvoMD Builder

## Prerequisites

Install node, npm and yarn on your local machine.

### To install Node and NPM

[Node LTS](https://nodejs.org/en/download/)
[NPM](https://docs.npmjs.com/about-npm-versions)

### To install Yarn

[Yarn](https://yarnpkg.com/getting-started/install)

### To check the versions of Node, NPM and Yarn

`node --version`
`npm --version`
`yarn --version`

## Local Environment Setup

### _SSH_

`git clone git@github.com:avoMD/web-app.git`

## Developing

### _1. In the project directory:_

Place .env file and run (Contact admin for .env file).

### _2. In Terminal run `yarn install`_

To install the project dependencies.

### _3. then run `yarn start`_

Runs the app in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.
You will also see any line errors in the console.
